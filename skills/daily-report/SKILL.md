# Daily Report
1. Create a daily report in Markdown format to summarize the development progress of the day.
2. Save it to a file named `daily_report_YYYY-MM-DD.md` in daily_reports directory of the project. 

## Format Requirements
Please strictly follow the Markdown format below:

```markdown
# 📅 Development Daily Report (YYYY-MM-DD)

## 💡 Summary
(Use one sentence to summarize today's main outcomes)

## 📝 Detailed Changes

### ✨ New Features
- [Module Name] Details ...

### 🐛 Bug Fixes
- [Module Name] Details ...

## ⚠️ Notes
- Any additional remarks or follow-ups
```
